## How to Upgrade

Follow these steps to ensure you run the latest version.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run `docs-ai --version` in your project directory.
  </Step>
  <Step title="Update Package" icon="download">
````bash
npm install docs-ai@latest
````

    Or use Yarn:
````bash
yarn upgrade docs-ai@latest
````
  </Step>
  <Step title="Verify Changes" icon="check-circle">
    Review the changelog and test your docs in preview mode.
  </Step>
</Steps>

<Expandable title="Advanced Upgrade Options" default-open="false">

For enterprise users, use the CLI flag for zero-downtime updates:

````bash
docs-ai upgrade --prod --backup
````

This creates a snapshot before applying changes.

</Expandable>
