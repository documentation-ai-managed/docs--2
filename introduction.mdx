## Overview

Documentation.AI empowers your team to build, publish, and maintain stunning documentation sites effortlessly. You leverage AI tools and reusable components to turn static manuals into interactive knowledge hubs. Get instant answers to user questions and streamline your documentation workflow today.

<Callout kind="info">
  Documentation.AI integrates seamlessly with your existing tools, supporting MDX, React components, and custom themes with your brand color `#8c8c8c`.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create your free account at `https://dashboard.example.com/signup`.
  </Step>
  <Step title="Create Project" icon="plus">
    Start a new site and choose your template.

````jsx
// Example project config
const config = {
  name: "My Docs",
  theme: { primaryColor: "#8c8c8c" }
};
````
  </Step>
  <Step title="Add AI Content" icon="zap">
    Use the AI assistant: "Generate an introduction page for my API docs."
  </Step>
  <Step title="Publish" icon="rocket">
    Deploy with one click to a custom domain.
  </Step>
</Steps>

<Tabs>
  <Tab title="npm" icon="package">
````bash
npm install documentation-ai-sdk
````
  </Tab>
  <Tab title="yarn" icon="package">
````bash
yarn add documentation-ai-sdk
````
  </Tab>
</Tabs>
